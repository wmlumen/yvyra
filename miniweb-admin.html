Publicación gratuita

La versión de producción puede publicar el HTML mediante una cuenta de servicio en Cloudflare Pages o Netlify. Los tokens deben permanecer exclusivamente en el servidor.

  1. El usuario completa la miniweb.
  2. El servidor genera archivos estáticos.
  3. Un trabajo de despliegue los publica.
  4. La plataforma guarda la URL resultante.
